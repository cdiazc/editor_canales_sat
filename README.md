# Intro
Editor de canales SDX 100% vibecodeado con gemini3 y opus4.5. La calidad del código es probablemente pésima pero funciona.

# Funcionalidades
- Editor de favoritos
- Reordenación y gestión sencilla
- Importación de paquetes desde KingOfSat

# Instalación y ejecución: 

Necesitas python3 instalado y tk

- MAC: brew install python-tk 
- ARCH: sudo pacman -S tk

En tu terminal:
$python editor_canales.py

# Notas

El Viark Combo probablemente ignora los nombres de las listas de favoritos al importar y solo los lee cuando se renombran manualmente desde el menú del deco. Es una limitación del firmware.
